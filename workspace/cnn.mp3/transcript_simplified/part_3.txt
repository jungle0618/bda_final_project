會不會有什麼 pattern是你要看一個圖片的左上角跟右下角才能夠找到的？也許沒有。如果沒有的話，這種 receptive field 就沒什麼用。我們之所以 receptive field 都是一個相連的領地，是我們覺得要偵測一個 pattern，那這個 pattern 就出現在整個圖片裡面的某一個位置，而不是分成好幾部分，出現在圖片裡面的不同位置。所以 receptive field 通常見到的都是相連的領地。 但如果你要設計很奇怪的 receptive field，去解決很特別的問題，那完全是可以的，這都是你自己決定的。 雖然 receptive field 你可以任意設計，但這邊要跟大家講一下最經典的 receptive field 的安排方式。 最經典的安排方式是什麼？第一個是會看所有的 channel。一般在做影像辨識的時候，你可能不會覺得某些 pattern 只出現在某一個 channel 裡面。所以會看全部的 channel。 既然會看全部的 channel，我們在描述一個 receptive field 的時候，只要講它的高跟寬就好，不用講它的深度，反正深度一定是考慮全部的 channel。 而這個高跟寬合起來叫做 kernel size。舉例來說，在這個例子裡面，我們的 kernel size 是 3 乘以 3。 但一般這個 kernel size 其實不會設太大，你在影像辨識裡面往往做 3 乘以 3 的 kernel size 就足夠了。如果你設 7 乘以 7 或 9 乘以 9，那算是蠻大的 kernel size，一般往往都做 3 乘以 3。 講到這邊你就會問，如果 kernel size 都是 3 乘以 3，意味著我們覺得在做影像辨識的時候，重要的 pattern 都只在 3 乘以 3 這麼小的範圍內就可以被偵測出來，聽起來好像怪怪的。有些 pattern 也許很大，也許 3 乘以 3 的範圍沒辦法偵測出來。等一下我們會再回答這個問題，我現在先告訴你，常見的 receptive field 設定方式是 kernel size 3 乘以 3。 一般同一個 receptive field 會有一組 neuron 去守備這個範圍。所以你劃一個 receptive field 以後，不會只有一個 neuron 去觀照它，不會只有一個 neuron 去守備它，往往會有一組 neuron 去守備它，比如說 64 個或者是 128 個 neuron 去守備一個 receptive field 的範圍。 到目前為止我們講的都是一個 receptive field。 各個不同 receptive field 之間的關係是什麼？你會把在最左上角的這個 receptive field 往右移一點，就製造一個新的守備範圍，製造另一個 receptive field。 這個移動的範圍，這個移動的量，叫做 stride。 像在這個例子裡面，stride 等於 2。stride 是一個你自己決定的參數，是你要自己調的 hyperparameter。但這個 stride 你往往不會設太大，往往設 1 或 2 就可以了。因為你希望這些 receptive field 之間是有重疊的。為什麼我們希望 receptive field 之間是有重疊的？因為假設 receptive field 完全沒有重疊，有個 pattern 正好出現在兩個 receptive field 的交界上面，就會變成沒有任何 neuron 去偵測它，你可能就會 miss 掉這個 pattern。所以我們會希望 receptive field 彼此之間有高度的重疊。 假設我們這個 stride 等於 2，第一個 receptive field 就在這邊，第二個就在這邊，第三個就再往右移兩格，放在這邊。 這邊遇到一個問題了，它超出了影像的範圍。 怎麼辦？有人可能會說，就不要在這邊排 receptive field。但你這樣就漏掉了影像邊邊的地方。如果一個 pattern 出現在邊邊的地方，就沒有 neuron 去觀照那些 pattern，沒有 neuron 去偵測出現在邊邊的 pattern。所以一般邊邊的地方也會考慮。 但超出範圍了怎麼辦？超出範圍就做 padding。什麼叫 padding？padding 是補 0。所以如果你今天 receptive field 有一部分超出影像的範圍之外，就當作裡面的值都是 0。其實也有別的補值方法，padding 就是補值的意思，也有別的補值方法。比如說有人會說，我這邊不要補 0，我補整張圖片裡面所有 value 的平均。可以。或者說我把邊邊的這些數字拿出來補，在這個沒有值的地方。可以。有各種不同的 padding 方法。 除了橫向移動，你也會有垂直方向上的移動。 在這邊，我們一樣垂直方向的 stride 也是設 2。所以你一個 receptive field 在這邊，垂直方向移動兩格，就有一個 receptive field 在這個地方。 你就會按照這個方式掃過整張圖片。所以整張圖片裡面每一寸土地都是有被某一個 receptive field 覆蓋的，也就是圖片裡面每一個位置都有一群 neuron 在偵測那個地方有沒有出現。
